# Microsoft.Data.SqlClient.Extensions.Azure 7.0 Releases

The following `Microsoft.Data.SqlClient.Extensions.Azure`
7.0 releases have been shipped:

> **Breaking change (.NET Framework only):** Starting with 7.0.2, the assembly's `AssemblyVersion` is `7.0.0.0` (previously `1.0.0.0`). On .NET Framework this changes the strong-name identity, so applications that replace this assembly without rebuilding must rebuild against 7.0.2 or add a binding redirect. Applications on .NET / .NET Core are not affected.
